My Learning Roadmap:

1. Solidify SQL + Data Modeling
   Goals: To practice advance SQL and understand the dimensional modeling
   Resources: 
     - "The Data Warehouse Toolkit" by Kimball (classic, but start with summaries online).
     - Mode Analytics SQL Tutorial (advanced sections on window functions, etc.).
     - Practice on platforms like LeetCode or StrataScratch.
       
2. Get familiar with Git
   Goals: Able to clone repo, commit changes, push & pull logics.
   Resources:
     - GitHub's "Hello World" guide or "Learn Git Branching" interative tutorial.
       
3. Dive into dbt
   Goals: Build a small dbt project from scratch.
   Steps:
     a. Sign up for a free dbt Cloud developer account.
     b. Connect it to a sample database (dbt provides a sample dataset, or use a public dataset in BigQuery's free tier).
     c. Follow the dbt Fundamentals course on the dbt Learn platform (free).
     d. Build a few models, write tests, and generate documentation.
   Project Idea: Take a messy public dataset (e.g., NYC taxi trips) and create a clean, well-modeled set of tables suitable for analysis. Document your decisions.

4. Learn Data Warehousing & Cloud
   Goals: Understand data lives eg: Snowflake, Bigquery, Redshifts,...
   Resources:
     - Cloud provider documentation (BigQuery free tier is great for practice).
     - Concepts like partitioning, clustering, and incremental models (dbt supports these).

  ***
  Create a GitHub repository with your dbt project. 
  Include a README explaining your modeling choices, and screenshots of the generated documentation. 
  This showcases your skills to employers.
